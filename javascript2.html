Day 5- Flexbox Practice


I've been trying to learn more about flexbox so today's lesson was great! The challenge was to make a kind of accordion style layout. Flex came in to center each of the elements in their certain place. I've spent way to much time trying center stuff using just CSS, so hopfully this saves me some time!


Takeaways


  • 1. There are a million ways to center things but flex seems so straight forward and intuative!
  • 2. The cubic-bezier tool came into action again. I'll definitly be using that more often!
  • 3. It turns out the include() method can be used to find elements that contain a certain string. We used it like this - e.propertyName.includes('flex') to find the elements that contained both flex AND flex-grow (IE used flex, while the rest use flex-grow).
