Day 1 - JS Keyboard Drummer


The first exersize starts out by mapping out the keys on a keyboard with some sweet drum sounds, creating a keyboard drummer. Being the weirdo I am I decided to switch out the stock sounds and pump in my favorite drum machine - the Roland 707.


Takeaways


  • 1. Datasets are always useful!
  • 2. Getting in the habit of using ES6 syntax is difficult (i.e. const) but I'm glad that they're tossed in the lesson.
  • 3. I'll be using keycode.info from now on! I guess I never realized that each key has a keycode! DUH

Day 2 - JS Clock


See the Pen JS-clock by Phil Gebauer (@philgeb) on CodePen.


For day 2 I created a pretty straight forward clock using some JS and CSS. I tried pausing the video and finishing myself, but found that I kept on relying on jQuery! This excersize was a great reminder of how powerful good ol' JavaScript is, and how important it is to keep practicing it!


Takeaways


  • 1. I relized that I tend to fall back on jQuery when it comes to manipulating the DOM.
  • 2. So there's a cubic bezier editor in Chrome dev tools..?! Sweet!
  • 3. Transform-origin is a cool little CSS tool that was used to change the origin of where the clock hands turned from. This was key for keeping them centered!
